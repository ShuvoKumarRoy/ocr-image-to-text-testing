# 📱 OCR Image to Text App Testing

## 📌 Project Overview
This project demonstrates manual testing of an OCR-based mobile application: **OCR Scanner: Image to Text**.

The goal was to evaluate functionality, usability, and performance of the application by identifying defects and validating features.

---

## 🎯 Objectives
- Validate OCR accuracy (English & Bangla)
- Test offline vs online functionality
- Identify UI/UX issues
- Perform functional and exploratory testing

---

## 🧪 Testing Scope
- Image to Text (OCR)
- Translate Feature
- Image to PDF
- Batch Scan
- Sharing & Export

---

## 🛠️ Tools & Technologies
- Manual Testing
- Microsoft Excel
- Android Device (Samsung Galaxy A30)
- Bug Tracking (Structured Reports)

---

## 📂 Project Files

### ✅ Test Cases
Located in `/Test_Cases`

### 🐞 Bug Reports
Located in `/Bug_Reports`
- 15 real defects identified
- Includes severity, steps, and impact

---

## 🔥 Key Findings
- OCR fails for Bangla in offline mode
- Handwritten text recognition is weak
- Critical bug: Edited text not saved
- Security issue: Premium credit reset loophole
- UX issues in keyboard handling and navigation

## 💡 Recommendations
- Improve offline language support
- Fix data persistence issues
- Add better UI feedback (toasts, alerts)
